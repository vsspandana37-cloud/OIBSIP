# OIBSIP
The Online Examination System is a web-based application developed to conduct exams digitally. It allows users to log in, attempt multiple choice questions (MCQs), and submit answers within a specified time limit. The system ensures a smooth and organized examination process.

 Features

User Login: Secure login for users to access the examination portal.

Profile Management: Users can update their profile details and change passwords.

MCQ Based Exams: Students can select and answer multiple choice questions.

Timer Functionality: The exam is conducted within a fixed time limit.

Answer Submission: Users can submit their answers after completing the exam.

Session Management: Proper closing of session and secure logout after the exam.


 Technologies Used

HTML

CSS

JavaScript

 Objective

The main objective of this project is to create a simple and efficient platform for conducting online examinations, reducing manual effort and making the process faster and more reliable.
